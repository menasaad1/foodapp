# el sawah - ملخص المشروع الكامل

## معلومات عامة عن التطبيق

**اسم التطبيق:** el sawah (السوق)
**النوع:** تطبيق توصيل الطعام الفاخر
**المنطقة:** دولة الإمارات العربية المتحدة
**المنصات:** تطبيق ويب (Web App) / تطبيقات الهاتف المحمول (iOS/Android)

---

## الميزات الأساسية

### 1. نظام المصادقة والحسابات
- تسجيل دخول آمن مع Supabase Auth
- إنشاء حساب جديد مع التحقق من البريد الإلكتروني
- إدارة كلمات المرور الآمنة

### 2. استعراض المطاعم والمأكولات
- عرض المطاعم المميزة على الصفحة الرئيسية
- فئات غذائية متنوعة (لبناني، هندي، إيطالي، إماراتي، إلخ)
- معلومات التقييمات والمراجعات
- أوقات التوصيل والرسوم

### 3. نظام السلة والدفع
- إضافة وحذف المنتجات من السلة
- حساب تلقائي للأسعار والتكاليف
- عملية دفع آمنة متعددة الخطوات
- طرق دفع متعددة (بطاقات، محافظ، كاش)

### 4. تتبع الطلبات الفورية
- خريطة تفاعلية توضح موقع الطلب والسائق
- مسار حالة الطلب مع تحديثات فورية
- معلومات السائق (الصورة، الاسم، التقييم)
- إشعارات فورية عند تحديثات الطلب

### 5. البحث والتصفية المتقدمة
- بحث نصي عن المطاعم والأطباق
- تصفية حسب الفئة والتقييم والسعر ووقت التوصيل
- اقتراحات ذكية أثناء الكتابة

### 6. المفضلة وسجل الطلبات
- حفظ المطاعم المفضلة
- عرض سجل جميع الطلبات السابقة
- إمكانية إعادة الطلبات السابقة بسهولة

### 7. إدارة الملف الشخصي
- تحديث المعلومات الشخصية
- إدارة عناوين التوصيل المحفوظة
- تعيين العناوين الافتراضية
- التفضيلات والقيود الغذائية

## البنية التكنولوجية

### Frontend
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **UI Components:** shadcn/ui
- **Icons:** Lucide React

### Backend
- **Database:** Supabase (PostgreSQL)
- **Authentication:** Supabase Auth
- **Real-time Updates:** Supabase Subscriptions
- **API:** Server Actions & Route Handlers

### الجداول الرئيسية
- `users` - بيانات المستخدمين
- `restaurants` - معلومات المطاعم
- `menu_items` - قائمة الطعام
- `cart_items` - عناصر السلة
- `orders` - الطلبات
- `order_items` - عناصر كل طلب
- `favorites` - المطاعم المفضلة
- `delivery_addresses` - عناوين التوصيل

## خطوات التشغيل

### 1. التثبيت
```bash
npm install
# أو
yarn install
```

### 2. إعدادات البيئة
- ضبط متغيرات Supabase في ملف `.env.local`
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

### 3. إنشاء قاعدة البيانات
```bash
# تشغيل السكريبتات SQL
# 01-schema.sql (إنشاء الجداول)
# 02-seed-data.sql (إدراج البيانات)
```

### 4. بدء التطبيق
```bash
npm run dev
# أو
yarn dev
```

### 5. الوصول للتطبيق
- افتح المتصفح على: `http://localhost:3000`
